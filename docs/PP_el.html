Πολιτική Απορρήτου UAB "Travel Union".

UAB Travel Union - Σεπτέμβριος 2020

Τελευταία ενημέρωση στις 20/10/2022

Αυτή η πολιτική απορρήτου διατίθεται σε εσάς από την Travel Union UAB, στη διεύθυνση Saulėtekio alėja 17, Vilnius, Lithuania ή στον ιστότοπό μας , εφεξής "TU" ή "εμείς". Συμμορφωνόμαστε με τη νομοθεσία περί προστασίας δεδομένων, όπως ο Γενικός Κανονισμός Προστασίας Δεδομένων της ΕΕ και οι τοπικοί κανόνες προστασίας δεδομένων και απορρήτου, που διέπουν την επεξεργασία προσωπικών δεδομένων που σας αφορούν και σας παρέχουν διάφορα δικαιώματα σε σχέση με τα προσωπικά σας δεδομένα. Επιπλέον, ακολουθούμε με συνέπεια τις βέλτιστες πρακτικές του κλάδου στην προστασία δεδομένων/απόρρητο και τις οδηγίες από τις αρμόδιες αρχές. Σκοπός αυτής της πολιτικής απορρήτου είναι να σας ενημερώσουμε για το πώς θα χρησιμοποιήσουμε τα προσωπικά σας δεδομένα που μας παρέχετε μέσω της εφαρμογής μας για κινητά (myTU App) σε σχέση με τις χρηματοοικονομικές υπηρεσίες που προσφέρει η TU, καθώς και προσωπικά δεδομένα που υποβάλλονται ή/και συλλέγονται μέσω των άλλων καναλιών μας, χρήση και μεταφορά προσωπικών δεδομένων σε τρίτους. Σας ενημερώνουμε επίσης για τα δικαιώματά σας, σύμφωνα με τους ισχύοντες νόμους περί προστασίας δεδομένων, σχετικά με την επεξεργασία των προσωπικών σας δεδομένων. Προτού υποβάλετε προσωπικά δεδομένα σε εμάς, σας συνιστούμε να διαβάσετε αυτήν την πολιτική απορρήτου, η οποία αποτελεί επίσης μέρος των όρων παροχής υπηρεσιών μας.

Εάν έχετε οποιεσδήποτε ερωτήσεις σχετικά με την πολιτική απορρήτου του TU ή θέλετε πρόσθετες πληροφορίες σχετικά με τον τρόπο άσκησης των δικαιωμάτων που αναφέρονται στο παρόν, μπορείτε να επικοινωνήσετε με τον Υπεύθυνο Προστασίας Δεδομένων του TU μέσω email στη διεύθυνση dpo@mytu.co. Πρόσθετες πληροφορίες επικοινωνίας είναι διαθέσιμες στην πλατφόρμα TU WEB και στην εφαρμογή για κινητά.

1. Τι είναι τα προσωπικά δεδομένα; Ασφάλεια δεδομένων

Δεσμευόμαστε να αποθηκεύουμε και να επεξεργαζόμαστε τα προσωπικά σας δεδομένα σύμφωνα με τις νομικές μας υποχρεώσεις. Το TU προσπαθεί να εξασφαλίσει κατάλληλα τεχνικά και οργανωτικά μέτρα για την προστασία των δεδομένων χρήστη και να παρέχει διαφανείς κανόνες προστασίας δεδομένων. Αυτή η πολιτική απορρήτου περιγράφει λεπτομερώς τους σκοπούς για τους οποίους επεξεργαζόμαστε τα προσωπικά σας δεδομένα, με ποιους τα κοινοποιούμε, ποια δικαιώματα έχετε σε αυτά τα δεδομένα και οτιδήποτε άλλο πιστεύουμε ότι είναι σημαντικό να γνωρίζετε.

Στην παρούσα Πολιτική Απορρήτου, "Προσωπικά Δεδομένα" σημαίνει κάθε πληροφορία που παρέχετε όταν αλληλεπιδράτε μαζί μας, όπως μέσω της εφαρμογής για κινητά TU, του ιστότοπου, της πλατφόρμας ιστότοπου (για νομικά πρόσωπα) ή καλώντας μας ή επικοινωνώντας μαζί μας μέσω άλλων πελατών κανάλια υπηρεσίας ή δεδομένα που συλλέγονται για εσάς μέσω της εφαρμογής ή του ιστότοπού σας για κινητά myTU και σας επιτρέπουν να αναγνωρίζεστε προσωπικά (π. π.χ. αριθμός τηλεφώνου) που μπορεί να χρησιμοποιηθεί για την αναγνώρισή σας ως άτομο. Ενδέχεται επίσης να συλλέξουμε προσωπικά δεδομένα για εσάς με άλλους τρόπους που σχετίζονται με τον λογαριασμό σας TU.

Σύμφωνα με τον Γενικό Κανονισμό Προστασίας Δεδομένων της ΕΕ αριθ. 2016/679 (GDPR), το TU είναι ο υπεύθυνος επεξεργασίας των προσωπικών σας δεδομένων. Ο υπεύθυνος επεξεργασίας δεδομένων είναι το πρόσωπο που καθορίζει για ποιους σκοπούς και με ποιον τρόπο θα υποβληθούν σε επεξεργασία τυχόν προσωπικά δεδομένα. Οι υπεύθυνοι επεξεργασίας δεδομένων πρέπει να θεσπίσουν πρακτικές και πολιτικές που συμμορφώνονται με τον GDPR.

Η διαδικασία επεξεργασίας προσωπικών δεδομένων που διεξάγεται από το TU μπορεί να περιγραφεί όχι μόνο στην παρούσα Πολιτική Απορρήτου, αλλά και στους Γενικούς Όρους και Προϋποθέσεις της Πλατφόρμας, Δηλώσεις Απορρήτου στην εφαρμογή για κινητά, Εσωτερικούς Κανόνες Επεξεργασίας Δεδομένων και στην ενότητα «Cookies» του Πολιτική Απορρήτου.

Όταν συλλέγουμε και χρησιμοποιούμε τα δεδομένα σας, έχετε το δικαίωμα:

  • Το δικαίωμα πρόσβασης στις πληροφορίες σας. Έχετε το δικαίωμα να ζητήσετε να σας στείλουμε ένα αντίγραφο όλων των προσωπικών δεδομένων που διατηρούμε για εσάς (με την επιφύλαξη ορισμένων εξαιρέσεων).
  • Το δικαίωμα σε ηλεκτρονικό αντίγραφο των στοιχείων σας. Μπορείτε επίσης να μας ζητήσετε να σας στείλουμε τις υποχρεωτικές πληροφορίες λογαριασμού που έχουμε για εσάς σε τυπική ηλεκτρονική μορφή ή να μας ζητήσετε να μεταφέρουμε αυτά τα δεδομένα σε τρίτο μέρος εάν το επιθυμείτε και εάν είναι τεχνικά δυνατό να το κάνουμε.
  • Το δικαίωμα να εναντιωθείτε στην επεξεργασία των πληροφοριών σας. Έχετε το δικαίωμα να αντιταχθείτε στην επεξεργασία τυχόν προσωπικών δεδομένων που επεξεργαζόμαστε όταν βασιζόμαστε σε έννομα συμφέροντα ως νομική βάση για την επεξεργασία (όπως ορίζεται στην ενότητα 3). Η ένστασή σας πρέπει να βασίζεται σε λόγους σχετικούς με την ιδιαίτερη κατάστασή σας.

Εάν υποβάλετε αίτημα άσκησης του δικαιώματός σας αντίρρησης, εάν έχουμε επιτακτικούς νόμιμους λόγους να συνεχίσουμε να επεξεργαζόμαστε τα προσωπικά σας δεδομένα, ενδέχεται να συνεχίσουμε να το κάνουμε. Διαφορετικά, θα σταματήσουμε να επεξεργαζόμαστε τα προσωπικά σας δεδομένα.

Ωστόσο, εάν αντιταχθείτε στη χρήση των προσωπικών δεδομένων που χρειαζόμαστε για να παρέχουμε τις υπηρεσίες, ενδέχεται να χρειαστεί να κλείσουμε τον λογαριασμό πληρωμών σας επειδή δεν θα μπορούμε να παρέχουμε τις υπηρεσίες.

  • Το δικαίωμα να μας ζητήσετε να μην σας διαφημίσουμε. Μπορείτε να μας ζητήσετε να μην σας στείλουμε άμεσο μάρκετινγκ. Μπορείτε να το κάνετε αυτό ακολουθώντας τις οδηγίες «κατάργηση εγγραφής» σε τυχόν μηνύματα ηλεκτρονικού ταχυδρομείου μάρκετινγκ.
  • Δικαίωμα αίτησης διόρθωσης ανακριβών στοιχείων. Έχετε το δικαίωμα να μας ζητήσετε να διορθώσουμε τα ανακριβή δεδομένα που διατηρούμε για εσάς. Εάν είμαστε πεπεισμένοι ότι τα νέα δεδομένα που έχετε παράσχει είναι ακριβή, θα διορθώσουμε τα προσωπικά σας δεδομένα το συντομότερο δυνατό.
  • Το δικαίωμα να ζητήσετε τη διαγραφή των δεδομένων σας. Έχετε το δικαίωμα να μας ζητήσετε να διαγράψουμε τα προσωπικά σας δεδομένα σε ορισμένες περιπτώσεις, για παράδειγμα, εάν έχουμε επεξεργαστεί τα δεδομένα σας παράνομα ή εάν δεν χρειαζόμαστε πλέον τα δεδομένα για τους σκοπούς που καθορίζονται στην παρούσα πολιτική απορρήτου.
  • Το δικαίωμα περιορισμού της επεξεργασίας των δεδομένων σας. Μπορείτε να μας ζητήσετε να περιορίσουμε την επεξεργασία των προσωπικών σας δεδομένων σε ορισμένες περιπτώσεις, για παράδειγμα εάν πιστεύετε ότι τα προσωπικά δεδομένα είναι ανακριβή και πρέπει να ελέγξουμε την ακρίβειά τους ή εάν δεν χρειαζόμαστε πλέον τα δεδομένα αλλά μας ζητάτε να τα διατηρήσουμε προκειμένου να ασκήσετε τα δικαιώματά σας.

Ο περιορισμός των προσωπικών σας δεδομένων σημαίνει ότι αποθηκεύουμε μόνο τα προσωπικά σας δεδομένα και δεν τα επεξεργαζόμαστε περαιτέρω εκτός εάν συναινείτε ή πρέπει να επεξεργαστούμε τα δεδομένα για να υποβάλουμε νομική αξίωση ή για να προστατεύσουμε ένα τρίτο μέρος ή το κοινό.

  • Δικαίωμα ανάκλησης της άδειας σας. Εάν μας έχετε δώσει τη συγκατάθεσή σας να χρησιμοποιήσουμε τα προσωπικά σας δεδομένα, μπορείτε να αποσύρετε τη συγκατάθεσή σας ανά πάσα στιγμή. Είναι νόμιμο για εμάς να χρησιμοποιούμε τα προσωπικά σας δεδομένα μέχρι να αποσύρετε τη συγκατάθεσή σας.
